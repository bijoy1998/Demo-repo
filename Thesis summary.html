Abstract

In recent years, research on deep learning has grown dramatically in text detection, digit recognition, and counter-detection. We propose an Automatic Meter Reading (AMR) system using YOLO-v7 for object detection and OCR for digit recognition. This system captures meter readings via webcam, processes them, and sends data to an IoT platform for real-time monitoring by users and power companies.

Chapter 1: Introduction

1.1 Background

In Bangladesh, many meters are still post-paid. Manual readings are time-consuming and error-prone. This project aims to automate the process using computer vision and deep learning.

1.2 Problem Statement

Consumers must manually send their meter readings. A camera-based solution with automatic processing can significantly reduce human involvement.

1.3 Objectives

  • To develop an AMR system using YOLO-v7.
  • To minimize human labor through automation.
  • To use IoT for real-time access by consumers and providers.

Chapter 2: Literature Review

Summarizes previous work on AMR, OCR, and image processing. Reviews segmentation-based and segmentation-free models, and compares tools like Tesseract vs CNNs.

Chapter 3: Methodology

  • Tools: Python, PyTorch, EasyOCR, Roboflow
  • Dataset: Paran√°-AMR (2239 images), augmented to ~4400 samples
  • Model: YOLOv7 pretrained on COCO, retrained for kWh/meter detection
  • OCR: EasyOCR for digit recognition
  • IoT: Data sent to a cloud platform for live monitoring

Chapter 4: Results

Counter detection: Precision: 91.2% (kWh), 91.4% (meter no); Recall: 92.4%, 89.7%

Digit recognition: Successfully applied EasyOCR on detected boxes.

Conclusion

This work demonstrated an efficient and accurate AMR system using YOLOv7 and OCR. It reduces labor and enables real-time usage tracking, offering a viable alternative for widespread implementation in regions with analog meters.
